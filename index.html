Intro to SpikeInterface

An introduction course for spikeinterface, aimed at users just getting started. Full YouTube playlist

If you watch these videos, you'll be able to make a simple spike sorting pipeline. The only prerequisit is that you have Python installed on your system. To do that, start here.

Some more advanced/specialist videos once you're happy with the core material. E.g. if you're comfortable with core lessons 1, 2, 3 you'll be able to tackle extra video 3A.
